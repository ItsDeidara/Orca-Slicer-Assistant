# Orca Slicer Assistant

A handy webpage to help you perform calculations related to the [orca slicer calibration steps](https://github.com/OrcaSlicer/OrcaSlicer/wiki/Calibration).

## Usage

You may simply clone this repo and open `index.html` in your browser. Alternatively, visit one of these web pages:

- https://dhnaranjo.github.io/Orca-Slicer-Assistant/
- https://orcalibrate.socuul.dev/
